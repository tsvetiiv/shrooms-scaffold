# SHROOMS SCAFFOLD SOLUTIONS

Shrooms Scaffold Solutions is a web application for scaffold rental, purchase, and custom scaffold requests.

## Features

* User registration and login
* Scaffold catalog
* Scaffold rental and purchase
* Custom scaffold requests
* Admin functionality
* Modern responsive UI

## Scaffold Types

* Facade scaffolds
* Mobile scaffolds
* Steel scaffolds
* Aluminium scaffolds
